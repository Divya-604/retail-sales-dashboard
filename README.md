 Blinkit Sales Dashboard
Project Overview

This project is an interactive sales dashboard built to analyze Blinkit's sales performance, outlet distribution, and product trends. It helps in understanding key business insights like total sales, item types, outlet size impact, and customer ratings.

-> Key Features
Total Sales Analysis ($1.2M+)
Average Sales & Ratings
Outlet Size & Location Analysis
Item Type Performance
Fat Content Distribution
Filter Panel for dynamic analysis
-> Dashboard Preview




-> Tech Stack
Excel
Data Visualization
Data Cleaning
-> Dataset

The dataset contains 500+ records including:

Item Type
Outlet Size
Outlet Location
Sales Data
Ratings
-> How to Use
Open the dashboard file (.pbix / .ipynb)
Use filters on left panel
Explore insights visually
-> Insights Gained
Tier 3 outlets have highest sales
Regular fat products dominate sales
Fruits & Snacks are top-performing categories
